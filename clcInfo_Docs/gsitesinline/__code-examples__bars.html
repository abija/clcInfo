Code examples \ Bars

Track some buff.

return BarAura("HELPFUL", "player", "Judgements of the Pure", nil, false, true)

Holy Power bar (for Cataclysm).

return true, nil, 0, 3, UnitPower("player", SPELL_POWER_HOLY_POWER)

A mana bar (make sure to use advanced settings and hide the icon)

local cm = UnitPower("player").
return true, nil, 0, UnitPowerMax("player"), cm, nil, cm

Track something like Beacon of Light, Earth Shield or Prayer of Mending.

return BarSingleTargetRaidBuff("Beacon of Light", false, true)

Track stacks from a buff.

local maxC = 3
local _ , _, _, c = UnitBuff("player", "Lightning Shield")
c = c or 0
return true, nil, 0, maxC, c, nil
