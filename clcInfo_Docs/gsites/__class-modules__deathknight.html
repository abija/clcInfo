Frost

First draft, based on: http://elitistjerks.com/f72/t105701-frost_dps_cataclysm_countdown/#Rotation

Icon code.

return IconFrost1()

Command line.

/clcinfo frost_mode single
/clcinfo frost_mode aoe

Switch between single target and aoe modes.

Functions to get information about the settings.

FrostMode()

Returns "single" or "aoe".



Unholy

First draft, based on: http://elitistjerks.com/f72/t105521-unholy_dps_cataclysm_ordinary_days/#Rotation

Icon code:

return IconUnholy1(diseaseClip)

diseaseClip: how many seconds before expiration to try and refresh diseases, defaults to 0.

Command line.

/clcinfo unholy_mode single
/clcinfo unholy_mode aoe

Switch between single target and aoe modes.

/clcinfo unholy_dt enabled
/clcinfo unholy_dt disabled

Toggle suggestion of Dark Transformation.

/clcinfo unholy_pest enabled
/clcinfo unholy_pest disabled

Toggle suggestion of Pestilence.

Functions to get information about the settings.

UnholyMode()

Returns "single" or "aoe".

UnholyDT()

Returns true or false.

UnholyPest()

Returns true or false.
