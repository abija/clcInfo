Code examples \ Icons

Spell cooldown.

return IconSpell("Avenging Wrath")

Spell cooldown that checks range of the spell.

return IconSpell("Judgement of Light", true)

Spell cooldown that checks if you are in melee range (as long as you have a melee range only ability).

return IconSpell("Judgement of Light", "Crusader Strike")

Spell that doesn't check range and is shown only when not on cooldown.

return IconSpell("Avenging Wrath", nil, "ready")

Buff on player.

return IconAura("HELPFUL", "player", 71187)

or

return IconAura("HELPFUL", "player", "Formidable")

Display the buff when it's active and the ability cooldown when it expires.

local visible, texture, start, duration, enable, reversed = IconAura("HELPFUL", "player", "Icy Veins")
if not visible then return IconSpell("Icy Veins") end
return visible, texture, start, duration, enable, reversed

Debuff on target.

return IconAura("HARMFUL", "target", "Stormstrike")

Debuff on target cast by the player.

return IconAura("HARMFUL", "target", "Stormstrike", "player")

Item that is equiped in slot 1 (head).

return IconItem(GetInventoryItemID("player", 1))

Internal Cooldown for Whispering Fang Skull (H).

return IconICD(71541, 45, 0, 1, 0.3)

Internal Cooldown for Ashen Band of Endless Vengeance

return IconICD(72412, 60, 0, 1, 0.3)

Display an aura when there are x seconds left on it.

local visible, texture, start, duration, enable, reversed = IconAura("HELPFUL|PLAYER", "player", "Judgements of the Pure")
if duration and (start + duration - GetTime()) <= 10 then
  return visible, texture, start, duration, enable, reversed
end

Display a buff when it's not active on you.

local name, _, icon = GetSpellInfo(89901)
if not UnitBuff("player", name) then return true, icon end

Show a spell when affected by a list of debuffs.

local visible = IconMAura("HARMFUL", "player", "Hamstring", "Frost Nova")
if visible then return IconSpell("Hand of Freedom") end

Show a spell when threat situation is dangerous.

local _, status = UnitDetailedThreatSituation("player", "target")
if status and status > 1 then
  return IconSpell("Hand of Salvation")
end
