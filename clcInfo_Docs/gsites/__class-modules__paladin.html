Retribution

Based on: http://elitistjerks.com/f76/t106276-retribution_concordance_4_0_1_a/#Rotation

Icon codes.

return IconRet1(csBoost [, useInq [, preInq]])
  • csBoost: if CS cooldown in seconds is lower than this value, it will be prioritized over next filler
  • useInq: if true, Inquisiton will be used
  • preInq: seconds left on Inquisition buff before trying to refresh at 3HP
  • On live you should only use first argument, csBoost.
  • Example: 
    return IconRet1(0.5)
return IconRet2()

Command line.

/clcInfo ret_fillers args
  • Change filler priority. Valid fillers: j, exo, how, hw, cons.
  • Example: 
    /clcInfo ret_fillers j exo how hw


Function reference

MBarSoV(alphaTarget, alphaRest [, showStack [, timeRight]])
  • alphaTarget: alpha for the bar displaying debuff on your current target
  • alphaRest: alpha for rest of the bars
  • showStack: specifies where the stack value is positioned
    • false/nil: hidden
    • "before": before name
    • not false/nil: after name
  • timeRight: if true, timer will be displayed on the right side in seconds
